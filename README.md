# vue-music

本项目是在入行前端时，跟着[黄奕](https://github.com/ustbhuangyi)大佬一起做的一个移动端音乐播放器实战项目

技术栈：vue2 + cubeUI

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
